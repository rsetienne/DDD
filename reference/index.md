# Package index

## All functions

- [`L2brts()`](https://rsetienne.github.io/DDD/reference/L2brts.md) :
  Function to convert a table with speciation and extinction events to a
  set of branching times
- [`L2phylo()`](https://rsetienne.github.io/DDD/reference/L2phylo.md) :
  Function to convert a table with speciation and extinction events to a
  phylogeny
- [`bd_ML()`](https://rsetienne.github.io/DDD/reference/bd_ML.md) :
  Maximization of the loglikelihood under the diversity-independent,
  possibly time-dependent diversification model
- [`bd_loglik()`](https://rsetienne.github.io/DDD/reference/bd_loglik.md)
  : Loglikelihood for diversity-independent diversification model
- [`brts2phylo()`](https://rsetienne.github.io/DDD/reference/brts2phylo.md)
  : Function to convert a set of branching times into a phylogeny with
  random topology This code is taken from the package TESS by Sebastian
  Hoehna, where the function is called tess.create.phylo
- [`conv()`](https://rsetienne.github.io/DDD/reference/conv.md) :
  Function to do convolution of two vectors
- [`dd_KI_ML()`](https://rsetienne.github.io/DDD/reference/dd_KI_ML.md)
  : Maximization of the loglikelihood under a diversity-dependent
  diversification model with decoupling of a subclade's diversication
  dynamics from the main clade's dynamics
- [`dd_KI_loglik()`](https://rsetienne.github.io/DDD/reference/dd_KI_loglik.md)
  : Loglikelihood for diversity-dependent diversification models with
  decoupling of a subclade from a main clade at time t = t_d
- [`dd_KI_sim()`](https://rsetienne.github.io/DDD/reference/dd_KI_sim.md)
  : Function to simulate a key innovation in macro-evolution with the
  innovative clade decoupling from the diversity-dependent
  diversification dynamics of the main clade
- [`dd_LR()`](https://rsetienne.github.io/DDD/reference/dd_LR.md) :
  Bootstrap likelihood ratio test of diversity-dependent diversification
  model
- [`dd_ML()`](https://rsetienne.github.io/DDD/reference/dd_ML.md) :
  Maximization of the loglikelihood under a diversity-dependent
  diversification model
- [`dd_MS_ML()`](https://rsetienne.github.io/DDD/reference/dd_MS_ML.md)
  : Maximization of the loglikelihood under a diversity-dependent
  diversification model with decoupling of a subclade's diversication
  dynamics from the main clade's dynamics
- [`dd_MS_loglik()`](https://rsetienne.github.io/DDD/reference/dd_MS_loglik.md)
  : Loglikelihood for macro-evolutionary succession under
  diversity-dependent diversification with the key innovation at time t
  = t_d
- [`dd_MS_sim()`](https://rsetienne.github.io/DDD/reference/dd_MS_sim.md)
  : Function to simulate the macro-evolutionary succession process
  assuming diversity-dependent diversification
- [`dd_SR_ML()`](https://rsetienne.github.io/DDD/reference/dd_SR_ML.md)
  : Maximization of the loglikelihood under a diversity-dependent
  diversification model with a shift in the parameters
- [`dd_SR_loglik()`](https://rsetienne.github.io/DDD/reference/dd_SR_loglik.md)
  : Loglikelihood for diversity-dependent diversification models with a
  shift in the parameters at time t = tshift
- [`dd_SR_sim()`](https://rsetienne.github.io/DDD/reference/dd_SR_sim.md)
  : Function to simulate the diversity-dependent diversification process
  with a shift in one or more of the parameters
- [`dd_loglik()`](https://rsetienne.github.io/DDD/reference/dd_loglik.md)
  : Loglikelihood for diversity-dependent diversification models
- [`dd_multiple_KI_loglik()`](https://rsetienne.github.io/DDD/reference/dd_multiple_KI_loglik.md)
  : Loglikelihood for diversity-dependent diversification models with
  multiple decoupling (rate shift) events
- [`dd_sim()`](https://rsetienne.github.io/DDD/reference/dd_sim.md) :
  Function to simulate the diversity-dependent diversification process
- [`optimizer()`](https://rsetienne.github.io/DDD/reference/optimizer.md)
  : Carries out optimization (finding a minimum)
- [`phylo2L()`](https://rsetienne.github.io/DDD/reference/phylo2L.md) :
  Function to convert phylogeny to a table with speciation and
  extinction events
- [`rng_respecting_sample()`](https://rsetienne.github.io/DDD/reference/rng_respecting_sample.md)
  : Sampling in which zero probabilities are removed
- [`roundn()`](https://rsetienne.github.io/DDD/reference/roundn.md) :
  Rounds up in the usual manner
- [`sample2()`](https://rsetienne.github.io/DDD/reference/sample2.md) :
  Takes samples in the usual manner
- [`simplex()`](https://rsetienne.github.io/DDD/reference/simplex.md) :
  Carries out optimization using a simplex algorithm (finding a minimum)
- [`td_sim()`](https://rsetienne.github.io/DDD/reference/td_sim.md) :
  Simulation of a diversity-dependent-like time-dependent process
- [`transform_pars()`](https://rsetienne.github.io/DDD/reference/transform_pars.md)
  : Transforming parameters from -Inf to Inf into parameters from -1 to
  1
- [`untransform_pars()`](https://rsetienne.github.io/DDD/reference/untransform_pars.md)
  : Untransforming parameters from -1 to 1 into parameters from -Inf to
  Inf.
